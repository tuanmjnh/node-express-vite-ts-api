# node-express-vite-ts-api

Generated template.

## Quick start

1. Install
```
npm install
```

2. Copy .env.example -> .env and fill values

3. Dev (hot reload)
```
npm run dev
```

4. Build + run
```
npm run build
npm start
```

API:
- /api/auth/login
- /api/auth/register
- /api/users
- /api/upload/single
